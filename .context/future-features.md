# 🔮 Planned / Future Features

To make this the ultimate developer Swiss Army Knife, the following tools are planned for future releases of the Developer Tools Collection.

## 🔄 Data & Format Converters
*   **JSON ↔ YAML Converter:** Seamlessly translate between JSON and YAML configurations.
*   **URL Encoder / Decoder:** Quickly encode or decode URI components and query parameters.
*   **XML / HTML Formatter:** Prettify and validate markup languages.

## 🌐 Web & Network
*   **Regex Tester:** Live regular expression matching engine with syntax highlighting.
*   **Cron Expression Generator:** Visually build and decode Cron schedules.
*   **HTTP/REST Client:** A lightweight, in-browser fetch tester for endpoints.

## 🛠️ Developer Utilities
*   **Unix Timestamp Converter:** Convert Unix timestamps to human-readable local/UTC dates.
*   **UUID / ULID Generator:** Instantly generate single or bulk v4 UUIDs.
*   **Hash Generator:** Generate MD5, SHA-1, SHA-256, and bcrypt hashes from text.
*   **Mock Data Generator:** Generate JSON arrays of placeholder data for UI testing.
